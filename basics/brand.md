Agencia · Web · Publicidad · Redes
Barlow Condensed
Display — Titulares y headlines
Condensed, impactante. Se usa en peso 800–900 para todos los títulos grandes. Permite apretar mucho texto en poco espacio con alta legibilidad. El estilo Italic se reserva para énfasis y citas.
Space Grotesk
UI — Body, labels, interfaces
Geométrica con personalidad. Sus terminaciones inesperadas le dan carácter sin perder legibilidad. Se usa en pesos 400/500/600 para cuerpo de texto, etiquetas y elementos UI.
Regla tipográfica
Nunca mezclar más de dos familias en una misma pieza. Barlow para impacto, Space Grotesk para información. Jamás usar itálica de Space Grotesk en titulares.
